# GoogleTranslateC
Google Translate library for C language.


# Depends
- libcurl4
- libcurl4-gnutls-dev
- libpthread


# Minimal Working Example
```sh
git clone https://github.com/ammarfaizi2/GoogleTranslateC;
cd GoogleTranslateC;
make run;
```
